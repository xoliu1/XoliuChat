# XoliuChatRoom

​    服务端用的连同一个局域网的电脑运行，即文件夹`ChatRoomClient(Client in IDEA)`内的文件，运行即可开放服务接口。

>  A simple ChatRoom in Android。只能用于局域网，app代码简单，是一个简单复习网络知识方面的demo。
